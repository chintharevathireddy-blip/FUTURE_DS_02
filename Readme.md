# Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer churn behavior using the Telco Customer Churn dataset. The aim is to identify factors affecting customer retention and provide recommendations to reduce churn.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Dataset
Telco Customer Churn Dataset from Kaggle.

## Key Insights
- Month-to-month customers have the highest churn.
- Electronic check users churn more frequently.
- Long-tenure customers are more loyal.
- High monthly charges increase churn probability.

## Recommendations
- Encourage yearly contracts using discounts.
- Improve onboarding experience.
- Offer retention campaigns for high-risk customers.
- Improve customer engagement strategies.
